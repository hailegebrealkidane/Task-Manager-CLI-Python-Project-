# Task Manager CLI

A simple Python command-line application to manage your daily tasks.

## Features
- Add tasks
- View tasks
- Mark tasks as complete
- Delete tasks
- Saves tasks locally using JSON

 Run the program:

python main.py
Usage
Follow the menu prompts:

Enter 1 to view tasks

Enter 2 to add a task

Enter 3 to complete a task

Enter 4 to delete a task

Enter 5 to exit

Requirements
Python 3.x
